# HW10.21.20TeamProfileGenerator

This application allows the user to create an html page that displays all members of a team.

The user begins the application which starts by asking what kind of employee to enter information about.

Depending on the answer, the last question is changed appropriately to get a specific piece of information.

The information collected is in a format of array with multiple objects in it.

This array is then utilized to generate a team.html page with all the information entered by the user displayed on the webpage.
